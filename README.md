# Interview Python AQA
Сборник конспектов и вопросов по Python, автоматизации тестирования и сопутствующим технологиям для подготовки к
собеседованиям на позиции Python AQA.

## Цели репозитория
- Систематизировать знания по Python core, ООП и устройству интерпретатора.
- Покрыть практические аспекты автоматизации тестирования: фреймворки, инфраструктура, CI/CD.
- Дать опорный материал по теории тестирования, процессам и soft skills для собеседований AQA.
